# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] -2025-10-29
### Added
- Initial public release of SecondBrain CLI
- Daily, Weekly, Backlog commands
- OpenTelemetry tracing and metrics
- Comprehensive unit and integration tests
- GitHub Actions CI pipeline
- Security, Contribution, and Code of Conduct policies
- Example configuration file (appsettings.example.json)

### Changed
- Configuration: appsettings.json made optional; environment variables recommended
- Project: removed local appsettings.json from source control

### Fixed
- Numerous worker bug fixes and performance improvements
